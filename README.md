## Getting started

1. Install dependencies with `npm install`.
2. Start the webpack development server with `npm start`.
3. Run the Jest test framework with `npm test`.
4. Run the Storybook visual testbed with `npm run storybook`.

## Dependencies

- axios
- classnames
- normalize.css
- react
- react-dom
- react-scripts
